# Mastering Java Statements

A personal study repository for the **Java Masterclass 2025: 130+ Hours of Expert Lessons** course on Udemy by **Tim Buchalka**.

## About this repository

This repo is used to refresh and reinforce Java knowledge through notes, code samples, and exercises from the course. Each folder under `sections/` corresponds to a specific section of the course.

## Current section

### 45. Mastering Java Statements, Whitespace, And Code Organization

This section covers the foundational art of writing clean, readable, and maintainable Java code. It dives into how statements are formed, how whitespace affects readability, and how to organize code for long-term maintainability.

**Location:** `sections/45_Mastering_Java_Statements_Whitespace_And_Code_Organization/`

**Key topics:**
- Java statements and how semicolons terminate them
- Whitespace, indentation, and line breaks for readability
- Brace style conventions and where to place curly braces
- Organizing code into packages, classes, and methods
- Meaningful naming and avoiding "magic" values
- Refactoring long or messy code into smaller, focused pieces
- Common formatting pitfalls and how to avoid them

**Why it matters:** Even if code is functionally correct, poor formatting and disorganization make it hard to read, debug, and extend. This section builds habits that keep code clean as projects grow.
